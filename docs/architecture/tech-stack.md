# Technology Stack

-   **Language**: Python 3.9+
-   **Web Framework**: FastAPI
-   **Server**: Uvicorn (ASGI)
-   **Data Processing**: Pandas
-   **Testing**: Pytest

## Frontend
-   **Library**: React 19
-   **Build Tool**: Vite 7.x
-   **HTTP Client**: Axios
-   **Node Version**: (Implied by environment, typically LTS)

## Infrastructure & DevOps
-   **Containerization**: Docker
-   **Orchestration**: Docker Compose
